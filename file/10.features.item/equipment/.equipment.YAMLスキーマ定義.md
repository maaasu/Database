# EQUIPMENT (装備) YAML スキーマ定義

武器、防具、アクセサリー、ツールなどの「装備して性能を変化させるアイテム」を表現します。
通常のアイテム共通項目（`schemaVersion` / `id` / `category` / `name` など）は `item.YAMLスキーマ定義.md` を参照し、本書では Equipment 固有項目のみ定義します。

## スキーマ定義
| キー                                    | 型            | 必須 | デフォルト | 説明                                                                      |
|:--------------------------------------|:-------------|:--:|:------|:------------------------------------------------------------------------|
| `equipment[].slot`                    | String       | ○  | -     | 装備スロット種別（後述）。                                                           |
| `equipment[].handType`                | String       | ×  | ONE   | 手持ち装備の手数。`slot=WEAPON` の場合に使用（`ONE` / `TWO`）。                           |
| `equipment[].requiredLevel`           | Integer      | ×  | 0     | 装備に必要なプレイヤーレベル。`0` で制限なし。                                               |
| `equipment[].requiredClasses[]`       | List<String> | ×  | -     | 装備可能クラスIDのリスト（任意）。未指定時は全クラス装備可。                                         |
| `equipment[].stats[]`                 | List         | ○  | -     | 装備中に適用される基礎ステータス補正のリスト（後述）。                                             |
| `equipment[].stats[].status`          | String       | ○  | -     | 対象ステータス（`StatusType`）。例: `ATTACK_POWER` / `DEFENSE` / `MOVEMENT_SPEED`。 |
| `equipment[].stats[].type`            | String       | ○  | -     | 補正方式（`FLAT` / `SCALAR`）。`FLAT` は加算、`SCALAR` は乗算係数。                      |
| `equipment[].stats[].value`           | String       | ○  | -     | 固定値 、範囲（例: `1` `1~4`）  補正値。`SCALAR` の場合は `0.10 = +10%`。                 |
| `equipment[].durability.max`          | Integer      | ×  | -     | 最大耐久値（任意）。指定しない場合は耐久管理なし。                                               |
| `equipment[].durability.consume`      | Integer      | ×  | 1     | 1回の使用/攻撃で減る耐久値。`durability.max` を指定した場合に使用。                             |

| キー                                    | 型            | 必須 | デフォルト | 説明                                                                      |
|:--------------------------------------|:-------------|:--:|:------|:------------------------------------------------------------------------|
| `maxStack`                            | Integer      | ×  | 1     | 最大スタック数。装備品は通常 `1` を推奨。                                                 |
| `equipment[].slot`                    | String       | ○  | -     | 装備スロット種別（後述）。                                                           |
| `equipment[].handType`                | String       | ×  | ONE   | 手持ち装備の手数。`slot=WEAPON` の場合に使用（`ONE` / `TWO`）。                           |
| `equipment[].requiredLevel`           | Integer      | ×  | 0     | 装備に必要なプレイヤーレベル。`0` で制限なし。                                               |
| `equipment[].requiredClasses[]`       | List<String> | ×  | -     | 装備可能クラスIDのリスト（任意）。未指定時は全クラス装備可。                                         |
| `equipment[].stats[]`                 | List         | ○  | -     | 装備中に適用される基礎ステータス補正のリスト（後述）。                                             |
| `equipment[].stats[].status`          | String       | ○  | -     | 対象ステータス（`StatusType`）。例: `ATTACK_POWER` / `DEFENSE` / `MOVEMENT_SPEED`。 |
| `equipment[].stats[].type`            | String       | ○  | -     | 補正方式（`FLAT` / `SCALAR`）。`FLAT` は加算、`SCALAR` は乗算係数。                      |
| `equipment[].stats[].value`           | String       | ○  | -     | 固定値 、範囲（例: `1` `1~4`）  補正値。`SCALAR` の場合は `0.10 = +10%`。                 |
| `equipment[].durability.max`          | Integer      | ×  | -     | 最大耐久値（任意）。指定しない場合は耐久管理なし。                                               |
| `equipment[].durability.consume`      | Integer      | ×  | 1     | 1回の使用/攻撃で減る耐久値。`durability.max` を指定した場合に使用。                             |
| `equipment[].onUse[].cooldownTicks`   | Long         | ×  | 0     | 使用アクション自体のクールダウン（tick）。`0` でクールダウンなし。                                   |
| `equipment[].onUse[].type`            | String       | ×  | Null  | 使用時(左クリック)の効果。                                                          |
| `equipment[].effects[]`               | List         | ×  | -     | 装備に紐づく追加効果リスト。`trigger` に応じて発動する（後述）。                                   |
| `equipment[].effects[].trigger`       | String       | ○  | -     | 発動タイミング（後述）。                                                            |
| `equipment[].effects[].type`          | String       | ○  | -     | 効果種別（`BUFF` / `Skill`）。                                                 |
| `equipment[].effects[].rate`          | Double       | ×  | 100   | 発動確率（`0〜100`）。`100` で必ず発動。                                              |
| `equipment[].effects[].skillId`       | String       | ×  | -     | 付与するBuff。`type=Skill` の場合に使用。※参照値                                       |
| `equipment[].effects[].buffId`        | String       | ×  | -     | 付与するBuff。`type=BUFF` の場合に使用。※参照値                                        |

### equipment[].slot
以下のいずれかの値を指定します。

- `WEAPON`
- `HEAD`
- `CHEST`
- `LEGS`
- `FEET`
- `ACCESSORY`
- `TOOL`

### equipment[].handType
`slot=WEAPON` のときに指定します。

- `ONE` : 片手装備
- `TWO` : 両手装備

### equipment[].stats[].type
- `FLAT` : 定数加算
- `SCALAR`: 乗算（ベース値に対して）

### 参照（ref）
Equipment の追加効果で Buff を参照する場合は `buff:` prefix を使用します（aliases: `bf`）。

- 例: `ref: buff:haste_small`

## YAML 例

### 例1: 基本的な武器

```yaml
schemaVersion: 1
id: bronze_sword
category: EQUIPMENT
name: "&fブロンズソード"
icon: IRON_SWORD
rarity: COMMON
lore:
	- "&7初心者向けの片手剣。"

maxStack: 1
equipment:
	slot: WEAPON
	handType: ONE
	requiredLevel: 3
	stats:
		- status: ATTACK_POWER
		  type: FLAT
		  value: 12
		- status: CRITICAL_CHANCE
		  type: FLAT
		  value: 0.03
    onUse:
        cooldownTicks: 80
        type: SWORD
	durability:
		max: 120
		consume: 1
```

### 例2: 使用時に特殊効果を発動する装備

```yaml
schemaVersion: 1
id: wind_crossbow
category: EQUIPMENT
name: "&bウィンドクロスボウ"
icon: CROSSBOW
rarity: RARE
lore:
	- "&7射撃時に風の加護をまとう。"

maxStack: 1
equipment:
	slot: WEAPON
	handType: TWO
	requiredLevel: 18
	requiredClasses:
		- RANGER
	stats:
		- status: ATTACK_POWER
		  type: FLAT
		  value: 28
		- status: ATTACK_SPEED
		  type: SCALAR
		  value: 0.08
	onUse:
		cooldownTicks: 80
		type: CROSSBOW_SHOT
	effects:
		- trigger: onHit
		  type: BUFF
		  rate: 30
		  buffId: 
            ref: buff:haste_small
		  durationTicks: 60
		- trigger: onUse
		  type: Skill
		  rate: 100
		  value: 
            ref: skill:wind_blast
```
