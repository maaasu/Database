# BUNDLE (パッケージ) YAML スキーマ定義

パッケージの基本的なスキーマ定義。

## スキーマ定義

| キー                          | 型      | 必須 | デフォルト | 説明               |
|:----------------------------|:-------|:--:|-------|:-----------------|
| `bundle[].lootTableId`      | String | ○  |       | LootTableId。※参照値 |
| `bundle[].onUse[].sound`    | String | ×  | Null  | 使用時に流れるサウンド      |
| `bundle[].onUse[].particle` | String | ×  | Null  | 使用時のパーティクル       |


## YAML 例

```yaml
schemaVersion: 1
id: magic_iron_ingot_bundle
name: &b魔法のパケット
icon: chest
rarity: UNCOMMON
lore:
  - &7魔力を帯びた珍しいパケット。
untradeable: false

bundle:
  lootTableId:
    ref: loot_table:magic_iron_ingot
  onUse:
    sound: block.anvil.land
    effect: block_break
```
